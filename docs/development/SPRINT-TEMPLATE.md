# LabCircle Sprint Planning & Retrospective Template

This document provides the standard markdown template for tracking sprint planning, execution progress, risks, issues, completed items, retro learnings, and planning for the subsequent sprint.

---

## 1. Sprint Header
*   **Sprint Name:** Sprint [Number] - [Short Goal Name]
*   **Duration:** [Start Date] to [End Date]
*   **Sprint Goal:** [Provide a clear 1-2 sentence description of what this sprint aims to achieve.]

*   `[x]` Completed tasks

- [ ] Task 2: Description
